Let me introduce myself

I am a Master Student currently conducting a Master's Research on AI in Universiti Teknologi Malaysia. When I was studying about AI, I noticed the capability and the beauty of Data Science, since then, I have begun to fall in love with Data Science.

For the past year, I have been studying all about Data Science by self-learning through all relevant resources and finally, I decided to start looking into building real end-to-end Data Science projects to solidify my knowledge. The projects that I have built utilize skills of data analysis, data visualization, machine learning, deep learning, and deployment.

Portfolio Projects

Smartwatch Usage Data Analysis
Data Analysis, Data Visualization

A case study done in the Google Data Analytics Certificate program. Smartwatch data was analyzed to provide high-level recommendations to the company. This project makes use of problem solving and data analysis skills to tackle a real-world scenario in a company.
The six phases of data analysis life cycle are incorporated into the case study. The life cycle consists of Ask, Prepare, Process, Analyze, Share, and Act, which begins from ascertaining the business task at hand, to providing analysis results and recommendations to the company.

Photo by Andres Urena on Unsplash

Anime Recommendation Website
Website, Docker, Web Scraping, Deep Learning

A recommendation system based on collaborative filtering and embeddings of user ratings was built using Tensorflow and Keras. Then, a website built with Flask was containerized and served with Nginx server using Docker and Docker Compose. The website was then deployed to a Google Compute Engine's Virtual Machine instance in Google Cloud Platform.
NOTE: The website is only accessible until Sept 2021 due to free trial.

SQL & Tableau Project
SQL, Data Visualization, Tableau

This project actually consists of three different parts. The first part used SQL queries to explore and analyze worldwide COVID-19 data, then the second part used some of the query results to transfer to Tableau for visualizations.
The third part then consists of SQL queries used for data cleaning, including but not limited to handling missing data and removing duplicates.
